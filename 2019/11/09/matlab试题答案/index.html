Posted on 2019-11-09 | In -随笔

一、矩阵运算
(1)A=[1,2,3;4,5,6;7,8,0];A(1,:)A(:,3)
ans=15
(2)A=[1,2,3;4,5,6;7,8,0];(A.A)’
ans= [1 16 49
4 25 64
9 36 0 ]

(3)A=[1,2,3;4,5,6;7,8,0];sum(A(:))
ans=36
(4)A=reshape(1:9,3,3);A(4,4)=111;A(:,5)=222
ans= [1 4 7 0 222
2 5 8 0 222
3 6 9 0 222
0 0 0 111 222]

二、求解常微分方程的MatLab命令
zeta=0.3;
[t,y]=ode45(@odefun,[0,30],[1;1],[],zeta);
plot(t,y(:,1));
function dy=odefun(t,y,zeta)
dy(1)=y(2);
dy(2)=1-y(1)-2zetay(2);
dy=dy(:);
end

三、利用rand函数构造5×5矩阵z
1.将z对角线元素清零
z=rand(5);
z=z-diag(diag(z));
2.将z中第二行除以2pi
z=rand(5);
z(2,:)=z(2,:)/(2pi);
3.将第一列与第五列相乘结果放在第一列
z=rand(5);
z(:,1)=z(:,1).z(:,5);

四、对边长5m的正方形铁板，在四个角减去四个相等的正方形以制成正方形无盖水槽，问如何使水槽容积最大。
fv=@(x)(-(5-2x)^2x);%匿名函数：当减去的正方形边长为x，其容积为-fv(x)
xmax=fminbnd(fv,0,2.5)
Vmax=-fv(xmax)

五、计算下表达式，n=99，x=[0,1]并按10等分取值。（不使用FOR、WHILE）
n=1:2:99;
x=linspace(0,1,10);
[xx,nn]=meshgrid(x,n);
z=nn.(-1).^((nn-1)/2)./((pinn).^2+4*xx.^2);
y=sum(z);
plot(x,y)

(1)在[10,30]上生成均匀分布的3阶随机矩阵的命令是
10+20rand(3)
(2)MatLab中求数值积分、常微分方程数值解的命令是
int,ode45
(3)分别写出矩阵的左除、点乘、转置的命令
\ . ‘
(4)MatLab中常用的逻辑关系操作符有

< == >= <= ~= & | ~
(5)求矩阵特征值和相应特征向量的命令是
[V,D]=eig(A)
(6)目前常用的四种滤波行为是：
低通、高通、带通、带阻

七、加窗是数据处理中常用的方式，列举出三种以上加窗方式，同时给出MatLab函数的调用格式(即添加长度n的信号窗，获得信号w)
矩形窗w=boxcar(n)
海明窗w=hamming(n)
汉宁窗w=hann(n)
布莱克曼窗w=blackman(n)
三角窗w=triang(n)
凯撒窗w=kaiser(n)

八、MatLab中快速傅里叶变换的函数为：fft，假设数组经过快速傅里叶变换得到结果y1，则y1的第一个点为直流分量，y1中其他点和原始信号幅度的关系为y1中其他点的值为原始信号幅度A的N/2倍，N为y1长度。

九、滤波
Fs=1000;
N=1000;n=0:N-1;
t=0:1/Fs:1-1/Fs;
Signal_Original=sin(2pi10t)+sin(2pi20t);
Noise_White=rand(1,1000);
Mix_Signal=Signal_Original+Noise_White;
对信号设计一个8阶截止频率为50Hz的巴特沃斯低通滤波器并进行滤波数据处理。
Wc=50/(Fs/2);
[b,a]=butter(8,Wc,’low’);
signal_filtered=filter(b,a,Mix_Signal);
plot(t,Mix_Signal,t,signal_filtered);

十、将[0,2pi]区间离散100个点，绘制y=xexp(-x)曲线，其中曲线为红色圆圈实线，并在图中标注x轴、y轴，标题为“曲线：xexp(x)”
x=linspace(0,2pi,100);
y=x.exp(-x);
plot(x,y,’ro-‘);
xlabel(‘x’);
ylabel(‘y’);
title(‘曲线：x*exp(x)’);
